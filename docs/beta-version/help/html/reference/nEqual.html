nEqual (boolean equal)

Syntax:

nEqual( expected : actual [: 'fieldName');

Description:

Compares the given Boolean values expected and actual. The assertion fails, if both values are different.

Return value:

void

Example: 

nEqual( *off : 1=2 );

// No exception message => OK.

nEqual( *on : 'Hello'='HelloWorld' );

// Never reached!
