clearAssertCounter (clear assertions counter)

Syntax:

clearAssertCounter();

Description:

Resets the counter of executed assertions.

Example: 

// Setup.

assertCallCnt = getAssertCalled();

// Execution.

assert( *on: *blank );
iEqual( 4: 2+2 );
aEqual( 'Hello': 'Hello' );

// Controls.

iEqual( assertCallCnt + 3 : getAssertCalled() );
clearAssertCounter();
iEqual( 0 : getAssertCalled() );
