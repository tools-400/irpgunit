assertEqual (all data types)

Syntax:

assertEqual( 'expected' : 'actual' [: 'message');

Description:

Compares the given values 'expected' and 'actual'. The assertion fails, if both values are different.

This assertion supports all data types.

Return value:

void

Example 1 (string): 

assertEqual( 'Hello' : 'Hello' );          

// No exception message => OK.        

assertEqual( 'HelloWorld' : 'Hello' );

// Never reached!

Example 2 (timestamp): 

assertEqual( z'2024-09-16-10.25.00.000' : z'2024-09-16-10.25.00.000' );          

// No exception message => OK.        

assertEqual( z'2024-09-16-10.25.00.000' : z'2024-12-31-00.00.00.000' );          

// Never reached!
