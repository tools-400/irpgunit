Upgrading From v4.0.4 to v4.1.0

Support for the following, old RPGUNIT libraries has been discontinued:

  • Version 1 released 22.04.2013
  • Version 2 released 10.10.2016

Furthermore the layout of the user space for carrying the test results to the Eclipse iRPGUnit plug-in has changed. The current version is 4. Version 3 user spaces are still supported.

Notice: This version requires iSphere v3.7.3 or higher, if you want to integrate it.
