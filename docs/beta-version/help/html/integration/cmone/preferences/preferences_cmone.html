Preferences CMOne Integration

The CMOne Integration preference page has options regarding to the integration of iRPGUnit into CMOne, which is a ChangeManagement Software of Task Force IT-Consulting GmbH, Germany.

This page is only available, when the optional iRPGUnit CMOne Integration feature has been installed.

Enable integration
