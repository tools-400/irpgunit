getMemberType (get member type)

Syntax:

srcType = getMemberType( 'file' : 'library' : 'member' );

Description:

Returns the source type of a given source member.

Return value:

Type of the specified member, e.g. RPGLE, DSPF, CLLE, etc.

Example: 

// Retrieves the member type of member ASSERT
// of file QINCLUDE in library RPGUNIT.

srcType = getMemberType('QINCLUDE': 'RPGUNIT': 'ASSERT');
