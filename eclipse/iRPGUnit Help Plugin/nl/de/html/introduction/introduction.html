Einführung

Das RPGUnit Plugin verbindet den IBM Rational Developer for i mit der RPGUnit Bibliothek auf einer IBM i, welche ursprünglich von Laction im September 2006 entwickelt worden ist. Es erweitert den Remote Systems Explorer um einen Eintrag im Kontextmenü, um RPGUnit Tests direkt aus dem Explorer heraus starten zu können.

Es basiert weitestgehend auf dem RPGUnit Plugin, welches ursprünglich von Mihael Schmidt für seinen Rpg Next Gen Editor entwickelt worden ist.

Auf der Server kommt das RPGUnit Testwerkzeug von "lacton" und "Cyril Clemenceau" in einer weiter entwickelten Form zum Einsatz. Leider scheint das RPGUnit Projekt nicht mehr aktiv zu sein. Trotzdem ist seine Internetseite immer noch ein guter Anlaufpunkt für Informationen zur Anwendung von RPGUnit.

Wichtig: Lesen sie die Hinweise zur Aktualisierung von RPGUnit, wenn sie auf iRPGUnit umsteigen möchten.
