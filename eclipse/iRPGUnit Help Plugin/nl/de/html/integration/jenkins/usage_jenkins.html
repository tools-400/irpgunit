Verwendung mit Jenkins

iRPGUnit Testfälle lassen sich von einer Jenkins Pipeline aus starten.

Die Installation von Jenkins geht über den Inhalt dieser Hilfe hinaus. Informationen dazu entnehmen Sie bitte der Jenkins Dokumentation.

Hinweis: Das Pipeline Stage View Plugin liefert eine schöne Darstellung der Jenkins Pipelines. Es sollte standardmäßig installiert sein, kann aber auch nachinstalliert werden.
