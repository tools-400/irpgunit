Verwendung mit VS Code

iRPGUnit ist in die VS Code Erweiterung IBM i Testing integriert. Testfälle lassen sich direkt aus VS Code heraus starten.

Installieren Sie die Erweiterung direkt vom VS Code Marktplatz:

Konfigurieren Sie die Erweiterung anhand der Dokumentation:

  • Installation
  • Writing Tests
  • Running Tests

Hinweis: Bei der Konfiguration ist es wichtig das in den Voreinstellungen die korrekten Werte der Attribute Test Source File und Test Source Directory angegeben sind. Anderfalls werden die Testfälle nicht in der Ansicht Test Explorer angezeigt.
