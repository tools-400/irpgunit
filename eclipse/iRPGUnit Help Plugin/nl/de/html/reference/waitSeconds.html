waitSeconds (wait n seconds)

Syntax:

waitSeconds( seconds [: 'nachricht' );

Beschreibung:

Hält den aktuellen Job für die angegebene Anzahl Sekunden an. Optional kann eine Statusnachricht ausgegeben werden. nach Ablauf der Wartezeit wird die ursprüngliche Statusmeldung wieder hergestellt.

Beispiel: 

// Warten für 5 Sekunden.

waitSeconds(5);          

// Warten für 3 Sekunden mit Statusnachricht.

waitSeconds(3 : 'Warten für 3 Sekunden ...');
