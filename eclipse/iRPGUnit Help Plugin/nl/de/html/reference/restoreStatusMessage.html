restoreStatusMessage (restore status message)

Syntax:

restoreStatusMessage();

Beschreibung:

Ersetzt die aktuelle Statusnachricht durch die zuvor angezeigte Nachricht.

Beispiel: 

// Anzeigen Statusnachricht.

displayStatusMessage('Hello World!');      
    
// Überschreiben Statusnachricht.

displayStatusMessage('The quick brown fox ...');  
        
// Wiederherstellen der 'Hello World!' Nachricht.

restoreStatusMessage();
