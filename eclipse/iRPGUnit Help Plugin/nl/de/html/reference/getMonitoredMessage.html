getMonitoredMessage (get monitored message)

Syntax:

msgInfo_t = getMonitoredMessage( *ON|*OFF );

Description:

Liefert die zuletzt aufgetretene *ESCAPE Nachricht aus dem Job Protokoll. Wird in der Regel im 'on-error' Abschnitt eines 'monitor' Blocks aufgerufen.

Example: 


monitor;                                             
  a = 10;                                            
  b = 0;     // Attempt made to divide by zero for   
  c = a / b; // fixed point operation. (MCH1211)     
  fail( 'Division by zero did not raise an error.' );
on-error;                                            
  msgInfo = getMonitoredMessage(*ON); // entferne Nachricht  
endmon;                               // aus dem Job Protokoll    
                                                     
aEqual( 'MCH1211': msgInfo.Id );
