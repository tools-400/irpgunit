assertJobLogContains (assert job log contains message)

Syntax:

assertJobLogContains( msgId : timeLimit );

Description:

Prüft, ob das Jobprotokoll eine gegebene Nachrichten ID im gegebenen Zeitraum JETZT und timeLimit enthält.
Der Wert von timeLimit sollte mit der Funktion getFullTimeStamp() gesetzt werden.

Beispiel: 

testStartTime = getFullTimeStamp();
// ausführen des Tests, von dem erwartet wird, dass er eine
// Nachricht MCH1211 dem Jobprotokoll hinzufügt
assertJobLogContains( 'MCH1211' : testStartTime );
