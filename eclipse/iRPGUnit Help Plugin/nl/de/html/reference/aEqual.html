aEqual (string equal)

Syntax:

aEqual( 'erwarteterWert' : 'tatsächlicherWert' [: 'fieldName');

Beschreibung:

Vergleicht die beiden Zeichenketten 'erwarteterWert' und 'tatsächlicherWert' miteinander. Der Testfall gilt als fehlgeschlagen, wenn die beiden Zeichenketten unterschiedlich sind.

Beispiel: 

aEqual( 'Hello' : 'Hello' );          

// No exception message => OK.        

aEqual( 'HelloWorld' : 'Hello' );

// Never reached!

Cobol Syntax:

Call "aEqual" Using
   By Content expected
   By Content actual
   [By Content "fieldName".
