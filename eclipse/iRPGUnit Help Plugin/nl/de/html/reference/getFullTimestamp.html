getFullTimestamp (get full timestamp)

Syntax:

getFullTimestamp();

Description:

Liefert die aktuelle Zeit als kompletten Zeitstempel, ohne Runden der Mikrosekunden wie es %timestamp() tut.

Funktionswert:

Liefert einen Zeitstempel im Format 'yyyy-MM-dd-hh.mm.ss.nnnnnn'.

Example 1: 

testStartTime = getFullTimeStamp();
