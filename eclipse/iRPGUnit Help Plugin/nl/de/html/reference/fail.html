fail (fail)

Syntax:

fail( 'nachricht' );

Beschreibung:

Erzeugt einen Fehler und bricht die Verarbeitung mit der angegebenen nachricht ab.

Beispiel: 

monitor;

  produceAnException(doFailure);
  
  fail('produceAnException(..) should have raised an error.');

on-error;
  // Success. Failure exception was seen.
endmon;

Cobol Syntax:

Call "fail" Using
   [By Content "message".
