Installation

Die Installation des iRPGUnit Plugins besteht aus drei Schritten:

  • Installieren des Plugins
  • Installieren der RPGUnit Bibliothek
  • Kompilieren der RPGUnit Bibliothek

Installieren des Plugins

Was sollte ich installieren?

Alles in der Kategorie iRPGUnit for IBM i gehört zu den Kernkomponenten des iRPGUnit Plugins und muss oder sollte installiert werden:

  • iRPGUnit Core (required)
  • iRPGUnit Help (recommended)

Features der Kategory iRPGUnit for IBM i (optional features) sind optional und können in Abhängigkeit ihrer Anforderungen installiert werden:

  • iRPGUnit Spooled File Viewer Feature (recommended, if iSphere is missing)

Features der Kategorie iRPGUnit for IBM i (optional integration features) sind optional und können in Abhängigkeit ihrer Anforderungen installiert werden:

  • iRPGUnit iSphere Integration (optional)
  • iRPGUnit CMOne Integration (optional)

Verwenden Eclipse Marketplace

Wählen sie vom Hauptmenü aus "Help > Eclipse Marketplace..." um zum Marketplace zu gelangen. Suchen sie nach 'iRPGUnit' and klicken sie auf den [Install] Button:

Wählen sie danach die zu installierenden Komponenten aus, klicken sie den [Confirm] Button und folgen sie den Anweisungen auf den nachfolgenden Seiten.

Verwenden der Update Site

Wählen sie vom Hauptmenü aus "Help > Install New Software...", um in das "Installieren" Fenster zu öffnen.

Ziehen sie dann die URL der Update Site in das "Arbeiten mit:" Feld:

Wählen sie danach die zu installierenden Komponenten aus, klicken sie den [Weiter] Button und folgen sie den Anweisungen auf den nachfolgenden Seiten.

Die URL der Update Site des iRPGUnit Plugins ist:

http://www.tools400.de/rpgunit/update/rdp8.0/

Verwenden eines lokalen Update Verzeichnisses

Gehen sie bei SourceForge auf die iRPGUnit Download Seite.

Führen sie danach folgende Schritte zum Erstellen eines lokalen Update Verzeichnisses aus:

  • Laden sie die Zip Datei auf ihren PC herunter.
  • Entpacken sie die Zip Datei in ein Verzeichnis ihrer Wahl. Dieses Verzeichnis wird dann ihr lokales Update Verzeichnis sein.
  • Wählen sie vom Hauptmenü aus "Help > Install New Software...", um in das "Installieren" Fenster zu öffnen.
  • Klicken sie den [Hinzufügen] Button um das Verzeichnis in die Liste der Update Sites aufzunehmen.
  • Verwenden sie den [Lokal...] Button zur Auswahl des Verzeichnisses.

Wählen sie danach die zu installierenden Komponenten aus, klicken sie den [Weiter] Button und folgen sie den Anweisungen auf den nachfolgenden Seiten.

Verwenden eines lokalen Update Archivs

Gehen sie bei SourceForge auf die iRPGUnit Download Seite.

Führen sie danach folgende Schritte zum Erstellen eines lokalen Update Archivs aus:

  • Laden sie die Zip Datei auf ihren PC herunter. Diese Zip Datei wird dann ihr lokales Update Archiv sein.
  • Entfernen sie gegebenenfalls die Versionsnummer aus dem Dateinamen, um den Namen für spätere Aktualisierungen stabil zu halten.
  • Wählen sie vom Hauptmenü aus "Help > Install New Software...", um in das "Installieren" Fenster zu öffnen.
  • Klicken sie den [Hinzufügen] Button um das Verzeichnis in die Liste der Update Sites aufzunehmen.
  • Verwenden sie den [Archiv...] Button zur Auswahl der Zip Datei.

Wählen sie danach die zu installierenden Komponenten aus, klicken sie den [Weiter] Button und folgen sie den Anweisungen auf den nachfolgenden Seiten.

Installieren der RPGUnit Bibliothek

Es gibt mehrere Wege, um die RPGUNIT Bibliothek zu installieren. Der einfachste Weg führt über den eingebauten Installationsdialog des iRPGUnit Plugins.

Verwenden des eingebauten Installationsdialogs

Die einfachste Weg zur Installation der RPGUNIT Bibliothek führt über der Installationsdialog, welcher von der Seite der iRPGUnit Benutzervorgaben aus gestartet werden kann. Voraussetzung hier ist, dass sie berechtigt sind eine FTP Verbindung von ihrem PC zu ihrem Server aufzubauen.

Dies sind die auszuführenden Schritte:

  • Öffnen sie die Benutzervorgaben über 'Fenster -> Benutzervorgaben'.
  • Filtern sie nach 'iRPGUnit'.
  • Klicken sie den [Hochladen] Button neben der Option 'Ausführungszeit: Produktbibliothek'.
  • Wählen sie die zu verwendende Verbindung aus.
  • Prüfen sie die FTP Portnummer. Der Standardport ist 21.
  • Prüfen sie den Namen der Bibliothek. Der Standardwert ist 'RPGUNIT'.
  • Prüfen sie den Namen der ASP-Einheit. Der Standardwert ist '*DEFAULT'.
  • Klicken sie den [Installieren] Button.

Sie müssen die Bibliothek nicht zwingend neu kompilieren. Dies ist nur dann erforderlich, wenn sie die RPGUnit Selbsttest ausführen wollen oder wenn sie einer anderen Namen als RPGUNIT für die Bibliothek verwenden möchten.

Übertragen der Sicherungsdatei mit 'upload_savf.bat'

Laden sie die iRPGUnit Zip Datei aus dem Abschnitt 'Files' bei SourceForge, e.g. iRPGUnit for RDi 8.0+ (v2.2.2.r Update Site).zip herunter.

Entpacken sie upload_savf.bat aus dem Ordner Server in ein leeres Verzeichnis auf ihrem PC.

Entpacken sie RPGUNIT.SAVF aus dem Ordner Server in den gleichen Ordner auf ihrem PC PC.

Starten sie upload_savf.bat mit folgenden Parametern: 

  upload_savf.bat hostName benutzerName passwort
Bei einem Aufruf von upload_savf.bat ohne Parameter werden Informationen zur Verwendung angezeigt.

Folgen sie danach den Anweisungen im Abschnitt Kompilieren der RPGUnit Bibliothek.

Übertragen der Sicherungsdatei mit per Hand

Erstellen sie zunächst eine leere Sicherungsdatei auf ihrem Server: 

Erstellen Sicherungsdatei:
  CRTSAVF FILE(QGPL/RPGUNIT)
Übertragen sie nun die Sicherungsdatei per FTP auf ihren Server: 
Öffnen einer DOS Box:
  XP, Win7: Start -> Execute -> cmd.exe

Starten einer FTP Verbindung zum Server:
  FTP your.server.com

Übertragen der Sicherungsdatei:
  binary
  put RPGUnit_v* QGPL/RPGUNIT
  
Beenden FTP Verbindung:
  quit

Folgen sie danach den Anweisungen im Abschnitt Kompilieren der RPGUnit Bibliothek.

Kompilieren der RPGUnit Bibliothek

Melden sie sich auf ihrem Server an, stellen sie die Bibliothek RPGUNIT aus der Sicherungsdatei wieder her und fügen sie die Bibliothek ihrer Bibliotheksliste hinzu: 

  RSTLIB SAVLIB(RPGUNIT) DEV(*SAVF) SAVF(QGPL/RPGUNIT)
  ADDLIBLE LIB(RPGUNIT)
Führen sie das Rexx Installationsscript A_INSTALL aus, um die RPGUnit Objekte zu erzeugen: 
  STRREXPRC SRCMBR(A_INSTALL) SRCFILE(QBUILD) PARM('install')

Falls sie sich für einen anderen Bibliotheksnamen entschieden haben, z.B. RPGUNIT3, müssen sie diesen wie folgt angeben: 

  STRREXPRC SRCMBR(A_INSTALL) SRCFILE(QBUILD) PARM('install RPGUNIT3')

Standardmäßig werden die iRPGUnit Objekte ohne Debuginformationen kompiliert. Fügen sie den Parameter list hinzu, wenn sie die Objekte mit Debuginformationen erzeugen möchten: 

  STRREXPRC SRCMBR(A_INSTALL) SRCFILE(QBUILD) PARM('install RPGUNIT list')

Fertig.

Vergessen sie nicht die Bibliothek RPGUNIT in die Bibliotheksliste ihrer Fernen Systeme aufzunehmen oder ändern sie die Produktbibliothek auf der Seite der iRPGUnit Voreinstellungen von *LIBL auf den tatsächlichen Namen ab.
