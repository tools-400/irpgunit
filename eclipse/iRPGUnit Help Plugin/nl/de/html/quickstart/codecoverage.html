Codeabdeckung

Das Aktivieren von Codeabdeckung (Code Coverage) kann zuweilen etwas mühsam sein. Leichter geht es mit folgenden Schritten:

  1. Kompilieren des Unit Test Serviceprogramms mit der Debug-Sicht *ALL.
  2. Aktivieren Debug Modus für das iRPGUnit Plugin vor dem Ausführen des Unit Tests.
    Der Debug Modus ist aktiv, wenn der kleine grüne Käfer, in der oberen rechten Ecke der RPGUnit View gedrückt ist.

  3. Setzen eines Serviceeingangs (SEP) auf das Unit Test Serviceprogram.
  4. Umschalten des SEPs auf Codeabdeckungsmodus:

  5. Ausführen Unit Test.
