Aktualisieren von v3.2.0 nach v4.0.0

Diese Version wurde für die Unterstützung von Cobol angepasst:

  • ... es gibt einen neuen RUCRTCBL Befehl.
  • ... es gibt einen neuen RUCRTRPG Befehl
  • ... der alte RUCRTTST Befehl ist als Kopie von RUCRTRPG weiterhin vorhanden.

Die folgenden RPGUnit Selbsttests wurden angepasst, weil von nun an alle von iRPGUnit erstellten Befehle formatiert werden. Diese Änderung war erforderlich, um das Testen von iRPGUnit zu vereinfachen:

  • CRTTSTT

Hinweis: Diese Version erfordert iSphere v3.7.3 oder höher, insofern iSphere eingebunden werden soll.
