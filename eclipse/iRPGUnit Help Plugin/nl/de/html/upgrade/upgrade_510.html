Aktualisieren von v5.1.0 nach v6.0.0

Verarbeitung von Zusicherungen

Optional können jetzt alle Zusicherungen eines Testfalls abgerabeitet werden. Bisher wurde die Abarbeitung der Zusicherungen beim Auftreten des ersten Fehlers beendet.

Siehe: Voreinstellungen

XML Ausgabeformat

Beginnend mit v6.0.0, ist der Standardwert für die XML Ausgabedatei auf *TYPE2 gesetzt. Das alte Format wird ausgegeben, wenn der Parameter XMLTYPE auf *TYPE1 gesetzt ist.

Diese Änderung war erforderlich, weil RPGUNIT jetzt den erwarteten und tatsächlichen Wert eines fehlgeschlagenen Testfalls ausgibt.
