Error: Duplicate iRPGUnit Objects

Symptom

Nachfolgend wird erklärt, warum folgende Fehlermeldung erscheint und wie das Prolbem behoben werden kann:

Grund

Der Grund für dieses Problem ist, dass das Startprogramm RUCALLTST oder RUPGMRMT ein anderes Serviceprogramm RUTESTCASE verwendet, als ihr Test Suite Serviceprogramm. Die kann beispielsweise passieren, wenn:

  • Das iRPGUnit Startprogram unter Angabe einer Bibliothek zum Binden des RUCALLTST Serviceprogramms mit CRTPGM erstellt worden ist.
  • Und ihr Test Suite Serviceprogramm die Bibliotheksliste zum Binden von RUTESTCASE verwendet.
    • Und die auf der Einstellungsseite angegebene Produktbibliothek sich von der in der Bibliotheksliste befindlichen Produktbibliothek unterscheidet.

Lösung

Für die Lösung des Problems existieren zwei Ansätze:

  1. Ändern sie die RSE Verbindung und ersetzten sie die dort angegebene iRPGUnit Bibliothek mit der Bibliothek auf der Einstellungsseite.
  2. Öffnen sie die Einstellungsseite und ändern sie die Produktbibliothek auf *LIBL.
