Integration with Jenkins

iRPGUnit Test Cases can be executed from a Jenkins Pipeline.

The installation of Jenkins is beyond the scope of this help text. For more information, please refer to the Jenkins documentation.

Notice: The Pipeline Stage View plug-in provides a nice visualization of the Jenkins pipelines. It should be installed by default but can also be installed separately.
