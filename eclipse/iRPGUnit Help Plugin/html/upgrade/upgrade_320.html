Upgrading From v3.2.0 to v4.0.0

This version has been changed to support Cobol unit tests. Please notice that:

  • ... there is a new RUCRTCBL command.
  • ... there is a new RUCRTRPG command.
  • ... there is still the old RUCRTTST command, which is a copy of RUCRTRPG.

The following RPGUnit self-tests have been changed, because all commands created by iRPGUnit are formatted, now. That change was necessary to ease testing iRPGUnit:

  • CRTTSTT

Notice: This version requires iSphere v3.7.3 or higher, if you want to integrate it.
