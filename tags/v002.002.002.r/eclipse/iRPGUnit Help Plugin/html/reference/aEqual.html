aEqual (string equal)

Syntax:

aEqual( 'expected' : 'actual' [: 'fieldName');

Description:

Compares the given String values 'expected' and 'actual'. The assertion fails, if both values are different.

Example: 

aEqual( 'Hello' : 'Hello' );          

// No exception message => OK.        

aEqual( 'HelloWorld' : 'Hello' );

// Never reached!
